```python
min()
```

- 매개변수 중 가장 작은 것을 반환
    - 숫자 : 가장 작은 수, 최소값
    - 문자 : 가장 앞 순서인 문자
        - A → z
        - 대문자 → 소문자
        - ㄱ → ㅎ
        - 자음 → 모음
        - 영어 → 한글
- 예시
    
    ```python
    min(5,-9,2,8,0,-1)
    # -9
    
    min('r', 'Q', 's', 'u', 'x', 'D', 'I', 'd')
    # D
    
    min('ㅇ', 'ㄷ', 'ㅕ', 'ㅔ', 'ㄴ', 'ㅜ')
    # ㄴ
    
    min('r', 'Q', 'x', 'D', 'c', 'ㅇ', 'ㅕ', 'ㅔ', 'ㄴ', 'ㅣ')
    # D
    ```
    

```python
max()
```

- 매개변수 중 가장 큰 것을 반환
    - 숫자 : 가장 큰 수, 최대 값
    - 문자 : 가장 뒤 순서인 문자
        - z → A
        - 소문자 → 대문자
        - l → ㄱ
        - 모음 → 자음
        - 한글 → 영어
    - 예시
        
        ```python
        max(5,-9,2,8,0,-1)
        # 8
        
        max('r', 'Q', 's', 'u', 'x', 'D', 'I', 'd')
        # x
        
        max('ㅇ', 'ㄷ', 'ㅕ', 'ㅔ', 'ㄴ', 'ㅜ')
        # ㅜ
        
        max('r', 'Q', 'x', 'D', 'c', 'ㅇ', 'ㅕ', 'ㅔ', 'ㄴ', 'ㅣ')
        # ㅣ
        ```
